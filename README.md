# LPU AutoLogin

**Auto-login script for Lovely Professional University Wi-Fi (Tampermonkey)**  
Encrypted storage, zero prompts — fast and safe for personal use.

---

## What this is
A Tampermonkey userscript that:
- Saves your LPU UID and password (encrypted in browser storage)
- Automatically fills & submits the LPU Wi-Fi login portal (`https://internet.lpu.in/*`)
- Requires **no passphrase prompts** — credentials are encrypted in storage and the encryption key is stored in your browser profile so login is automatic
- The script does **not** send any data anywhere
